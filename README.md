# Portfolio Website

Personal portfolio site for Kavya Casula.

## Status

🚧 Early planning — nothing built yet. See [CLAUDE.md](CLAUDE.md) for how this
project is developed collaboratively with Claude Code.

## Goals

- A fast, clean personal site showcasing projects, skills, and experience
- Easy to maintain: editing content shouldn't require touching layout code
- No heavyweight tooling unless the project actually needs it

## Prior art

A throwaway single-file demo lives at `~/claudecode/portfolio-demo/index.html`
(dark theme, hero / projects / skills / experience / contact sections). It can
serve as a visual starting point or be discarded entirely.

## Local development

Open `index.html` directly in the browser, or run a local server:

```bash
python3 -m http.server 8000
```

Chrome on this machine is installed as **"Google Chrome 2.app"**, so from the
terminal: `open -a "Google Chrome 2" <file-or-url>`.
